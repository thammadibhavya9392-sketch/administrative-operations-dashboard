# Administrative Operations Dashboard

## 📌 Project Overview

This project demonstrates an Administrative Operations Dashboard developed using Microsoft Excel. It helps manage employee records, attendance, meetings, and expenses while providing management with key performance indicators (KPIs) through an interactive dashboard.

---

## ✨ Features

- Employee Master Database
- Attendance Tracking
- Meeting Scheduler
- Expense Tracker
- KPI Dashboard
- Charts & Data Visualization
- Administrative Reporting
